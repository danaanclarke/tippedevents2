How does CrowdTipped work?

  1. Propose an event.
  2. Set a tipping point, which is the number of people that need to commit before the event goes live.
  3. Get a notification when the event is tipped.

That's it!

Still don't get it?

Let's say you want to play basketball with some friends, and you want to make sure you get enough for at least 3 on 3, or 6 people total. CrowdTipped allows you to set a tipping point on the event, meaning that only if 6 people have said they'll be there will the event get tipped. Think of it like Kickstarter for events.
